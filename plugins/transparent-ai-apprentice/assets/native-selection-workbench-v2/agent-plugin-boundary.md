# 宿主 Agent 插件边界

本交付是依附于当前宿主 Agent 的 UI 插件层，不是独立 AI 产品，也不是第二个智能体运行时。

## UI 负责

- 读取并展示宿主软件传入的原生选区、对象、定位器与只读上下文。
- 采集老师的局部修改意见、约束、保护范围与审核决定。
- 展示当前宿主 Agent 返回的读取、预览、等待审核、执行、错误、离线和回滚状态。
- 将可审计的选择与意见封装为 `MingTuOverlay.packet()`，交回当前宿主 Agent。

## UI 不负责

- 不在浏览器内运行模型或自主规划。
- 不请求、保存或代理任何 API Key。
- 不调用独立模型 API，也不绕过当前宿主 Agent 直接调用工具。
- 不把本地 Canvas 示意、文字差异或执行预览宣称为技术验收。
- 不自动执行、接受规则或解除包装投产门禁。

桌面信标是宿主 Agent 的上下文入口和状态指示器。其状态来自宿主 Agent/宿主桥；它不拟人化宣称能力，也不遮挡核心工作区。
