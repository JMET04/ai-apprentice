# AI 学徒 1.0.0

本版新增自然沟通能力：澄清、纠错、失败反馈和进度说明会先回应用户真正说的那句话，再给出最小可行动作。系统会检查客服腔、流程旁白、空泛安慰、过度迎合、假装真人和情感依赖，同时保留技术事实、证据和人工审核边界。

看得见成长，教得会做事。

这是 AI 学徒的首个公开版本。它将“人教 AI 做事”的核心闭环、包装设计的 Image2 到 CAD 流程、老师蒙版纠错台和完整 AICAD 1.2.0 集成放入同一个可验证插件包。

本次产品化补强已经接通真实纠错任务的提交、存储、老师审核、失败重试和结果回放；首个 AICAD 工程适配器可对审核后的尺寸对象执行局部修改、生成 AICAD / AutoCAD 脚本 / DXF 产物并保留精确回滚点。默认 MCP 入口收敛为 7 个老师工具，进阶入口为 30 个任务工具，内部维护路由不再干扰普通用户。

## 本版重点

- 八阶段包装流程：需求澄清、深度方案、Image2 中文样图、自查、蒙版纠错、局部修改、AICAD 制图和最终人工复核。
- Image2 首次提示词优化：先路由现有能力，再把确认事实、约束、负面提示和检查项编译成可验证指导包；本机专业提示词库可选增强，离线也能工作。
- 自然沟通：按澄清、纠错、失败、进度、技术、情绪和轻松交流场景生成表达指导，并在发送前检查常见 AI 腔与关系边界。
- AI 学徒审校台：自由画笔、圈选、框选、箭头、文字、撤销重做、草稿恢复、缩放、显隐、只读回放、桌面与手机触控。
- 精准局部修改：恢复原工程图片蒙版，Word / Excel 文字与工程软件对象分别使用两个独立工作台；默认禁止因一个小问题重做全部内容。
- Word / Excel 点改：Word 绑定段落，Excel 绑定单元格；源文件不覆盖，未选中 OOXML 部件逐项验哈希。
- 复杂 Office 点改：支持跨文字 run、Word 表格段落、Excel 合并单元格锚点和富文本共享字符串；公式和非锚点修改继续失败关闭。
- 真实任务闭环：蒙版提交会进入持久任务库，支持去重、老师审核、重试、执行结果和状态回放，离线提交会明确排队而不冒充已保存。
- 工程对象执行：审核后的 AICAD 尺寸修改只触碰选中对象，并输出哈希、审计、清单与精确回滚副本。
- 多轮学习验证：未经老师批准的规则不会影响下一次执行；批准后可稳定复用，停用后立即停止应用。
- 规则差异决策：按上下文、具体性和老师例外做出可解释选择，并显式标出存在疑点的位置。
- 严格 CAD 交接：相对路径、媒体类型、SHA-256、尺寸权威、会话绑定、请求绑定、输出范围和结果哈希。
- 完整 AICAD 1.2.0：21 条包装规则、确定性二维编译、SolidWorks 三维候选流程和宿主证据边界。
- 结构化学习：执行轨迹、老师纠错、停用规则草稿和带来源的 RAG 证据包。

## 重要边界

- Image2 只负责视觉候选和局部修改，像素不能作为工程尺寸。
- 自动检查、RAG 或 CAD 输出不代表技术验收或量产批准。
- 老师纠错只形成默认停用规则草稿，不自动启用规则。
- AutoCAD 与 SolidWorks 的历史宿主证据不会冒充本次发布执行；真实环境仍需人工复核。

## 安装

下载 `transparent-ai-apprentice-codex-plugin.zip`，或从源码运行：

```powershell
npm ci
npm run verify:plugin
npm run package:codex-plugin
npm run install:codex-plugin
```
